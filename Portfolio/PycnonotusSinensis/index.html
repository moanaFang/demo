翁翁保護局

白頭翁不吃小米

也不吃穀類、米飯、鮮奶、土司麵包跟餅乾糖果巧克力。

多樣化食物

白頭翁是雜食性動物，以果實、昆蟲為主食，無法消化小米、穀類，也不能吃含有人工添加物（鹽、糖、為素等等）或加工過的人類食物。野翁的食物來源很豐富，若要讓家翁長得健康漂亮就要學習野翁的菜單，給予多樣化的食物。

飼料

應避免單一飼料餵食，混料營養較豐富

  • 常見飼料：合美健6號 / 合美健21B(嗜蟲性) / 凡賽爾UNI / 凡賽爾F16 / 路比爾水果滋養丸
    • 合美健的飼料價格較便宜，是凡賽爾及路比爾的1/4左右
    • 凡賽爾及路比爾的顏色繽紛，但大多數的翁會挑顏色吃，且吃過以後會不屑吃合美健，換飼料前要衡量自己的口袋深度XD
  • 換新款飼料時應採逐天增加比例混料的方式更換，避免不適應的狀況
  • 混料一般以嗜蟲+嗜果+高蛋白三種混合為主，比例則無需太嚴謹，可以照幼鳥嗜蟲成鳥嗜果來配重

水果

常溫、無農藥殘留的水果

  • 首選木瓜，是幼鳥預防軟腳的好幫手
  • 最好輪換不同的水果以補充不同營養
  • 酪梨是油脂不是水果，不能給鳥吃
  • 要退冰，太冰的水果對鳥的腸胃不好
  • 要洗乾淨，皮最好都剝掉或削掉，避免農藥殘留
  • 可切丁或磨泥，柳橙類可以直接榨汁
  • 夏天要注意水果不能放太久，會臭掉
  • 不要給鳥吃人咬過的水果，人類口腔的細菌比鳥骯髒千萬倍，這樣餵食有可能會造成鳥生病

麵包蟲

  • 鳥店就可以買到，一兩的麵包蟲就可以吃一陣子了
  • 給幼鳥的麵包蟲要把頭剪掉或者先溺死，避免蟲的口器破壞幼鳥口腔腸胃
  • 成鳥會自己把蟲敲死或咬爛才吃下去，就不需要自己殺蟲了
  • 蟲粉是個只能騙騙幼鳥的東西，成鳥大多不愛XD
  • 麵包蟲內含的磷質可能會影響鈣質吸收，若是需要補鈣的母翁建議不要吃太多麵包蟲

飲水

  • 翁翁很愛喝水，需要每天勤更換，不要讓他喝到不乾淨的水，會生病的。
  • 蜂蜜水鳥會很愛，比例不要調太濃會太甜（建議1：10或更淡）

蔬食

  • 不做任何調味的南瓜或地瓜蒸熟後放涼再給，鳥不能吃太冰也不能吃太燙的東西
  • 葉菜類確定洗淨無農藥殘留才能給
  • 野外白頭翁健胃整腸食品：芸香科刺蔥與臭辣木種子或接骨木科的葉片（不能消化但表面油脂可保持腸道健康）
  • 左手香汁

錯誤飲食

NG食物清單

  • 太冰或太熱的食物
  • 人工調味的食物
  • 酪梨
  • 單一飲食
  • 穀類、小米
  • 髒水
  • 人類口水沾過的食物

餵食幼鳥的注意事項

主食(濕料)

  1. 濕料：野鳥飼料用常溫水泡軟，與木瓜泥攪拌成稠狀餵食
    • 夏天需至少每2小時換一次，避免臭酸
    • 軟腳是幼鳥最怕遇上的症狀（嚴重時有致命的可能），木瓜則是預防軟腳最好的秘方，最好每天給予或補充維他命[需要鳥專科醫生確定劑量]
  2. 麵包蟲：溺死或把頭剪掉再餵，避免蟲的口器傷到幼鳥內臟。
    • 蟲也是幼鳥很重要的營養來源，如果不敢餵的話可以改買蟲乾或補充合美健21B嗜蟲性飼料
  3. 水：乾淨的常溫飲用水，需每天更換。

器具

可於鳥店購買幼鳥餵食器，也可利用 吸管+竹筷 或 利樂包吸管+牙籤 自制。藥局的3ml~5ml針筒拿掉針頭請店家回收也是很好的餵食器

  • 吸管將濕料吸起，竹筷/牙籤從吸管末端插入，將濕料推入幼鳥口中
  • 直接推至喉嚨處讓鳥直接吞下(放太淺的話有可能會噎到)

多久餵一次？

  • 幼鳥發育正是最需要營養的時候，只要幼鳥開始哭夭討食就要餵，餵到他不再有討食行為就停止，靜待下一次哭夭來臨。(約15-30分鐘)

要餵到多大？

  • 大約要餵到變成亞成鳥的時候（尾巴長出來、快要跟身長一樣時）
  • 如果發現小翁開始有啄食的行為的話，可以放濕料給他自己啄看看（但還是要持續餵食）
  • 過渡期是乾料跟濕料都放，讓他學著自己吃，但還是需要餵食以避免吃的不夠營養不足
  • 等到他乾料都可以自己吃的時候就不用再像奶媽一樣跟前跟後餵食了

參考資料

  • 血色素沉著症
  • 全陽林醫師：鵯科的鳥小時候肉食性吃蟲，長大偏素食嗜果
